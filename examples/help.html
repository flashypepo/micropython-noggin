Simple file operations

  • GET /disk to get filesystem information
  • GET /disk/free to get information about available disk space
  • GET /file to list available files
  • GET /file/<path> to get a file (for example, boot.py)
  • PUT /file/<path> to write a file
  • POST /file/<path> to rename a file (new name is POST body)
  • DELETE /file/<path> to delete a file

Non-file related

  • GET /mem/free for available memory
  • GET /net/eth0 for information about the wireless client interface
  • GET /net/eth1 for information about the wireless ap interface
  • GET /reset to reset the board by calling machine.reset

For either of the /net/... enpoints above, you can request a specific key by adding another component to the URL. For example, GET /net/eth0/mac.
